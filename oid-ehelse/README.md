# oid-ehelse

Maskinlesbar liste (JSON) over OID'er som [Direktoratet for e-helse](https://www.ehelse.no/) forvalter, 
inkludert rot-OID'er som er tildelt andre. 
Utvikling, skal publiseres annet sted, så ikke bruk denne.
